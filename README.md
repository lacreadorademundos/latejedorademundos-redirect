# latejedorademundos-redirect

Redirect from [latejedorademundos.com](https://latejedorademundos.com) to [www.lacreadorademundos.com](https://www.lacreadorademundos.com)

## How it works

- `index.html` — Meta-refresh + JS redirect preserving path, query and hash
- `404.html` — Same logic, served by GitHub Pages for unknown routes
- `CNAME` — Maps to `latejedorademundos.com`
